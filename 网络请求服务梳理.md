# 网络请求服务梳理

## 一、主页相关

### 1. NLRecommendAlbumListService
- **路径**: `Networking/网易云API/为你推荐/NLRecommendAlbumListService.h/.m`
- **API**: 网易云 `/personalized`（个性化推荐）
- **方法**: `fetchRecommendPlayListWithLimit:success:failure:`
- **返回**: `NSArray<NLRecommendAlbumListModel *>`
- **用途**: 获取「为你推荐」歌单列表（根据用户口味个性化推荐）

### 2. NLChineseSongListService
- **路径**: `Networking/网易云API/为你推荐/NLChineseSongListService.h/.m`
- **API**: 网易云 `/top/playlist/highquality`（精品歌单）
- **方法**:  
  - `fetchRecommendPlayListWithLimit:success:failure:` 华语精品（cat=华语）
  - `fetchHighQualityPlaylistsWithCategory:limit:success:failure:` 按分类获取，cat 如 "华语"、"欧美"、"流行"
- **返回**: `NSArray<NLRecommendAlbumListModel *>`
- **用途**: 获取精品歌单（华语、欧美、流行等）

### 3. NLSingerAlbumListService
- **路径**: `Networking/网易云API/JJAlbum/NLSingerAlbumListService.h/.m`
- **API**: 网易云 `/artist/album`（歌手专辑列表，当前写死 id=3684 林俊杰）
- **方法**: `fetchSingerListWithSuccess:failure:`
- **返回**: `NSArray<NLSingerAlbumListModel *>`
- **用途**: 获取某歌手的专辑/歌单列表（大卡片分区数据）

---

## 二、歌单/专辑详情

### 4. NLSongListService
- **路径**: `Networking/网易云API/歌单详情/NLSongListService.h/.m`
- **API**: 网易云歌单详情接口
- **方法**: `fetchPlayListDetailWithId:completion:`
- **返回**: `NLHeaderModel` + `NSArray<NLListCellModel *>`
- **用途**: 获取歌单详情（封面、名称、歌曲列表），用于歌单详情页

### 5. NLAlbumService
- **路径**: `Networking/网易云API/专辑/NLAlbumService.h/.m`
- **API**: 网易云专辑详情接口
- **方法**: `fetchAlbumDetailWithId:completion:`
- **返回**: `NLHeaderModel` + `NSArray<NLListCellModel *>`
- **用途**: 获取专辑详情（封面、歌曲列表），用于专辑详情页

### 6. NLPlaylistService
- **路径**: `Networking/网易云API/歌单详情/NLPlaylistService.h/.m`
- **API**: 网易云 `/top/playlist/highquality`（精品歌单，支持分类）
- **方法**: `fetchPlaylistsWithCategory:success:failure:`
- **返回**: `NSArray<NLPlaylistModel *>`
- **用途**: 按分类获取精品歌单列表（如搜索页「该品类精选歌单」），可传 `cat` 筛选

---

## 三、播放与评论

### 7. NLSongService
- **路径**: `Networking/网易云API/音乐播放Url/NLSongService.h/.m`
- **方法**: `fetchPlayableURLWithSongId:success:failure:`
- **用途**: 获取歌曲可播放的 URL

### 8. NLCommentService
- **路径**: `Networking/网易云API/评论/NLCommentService.h/.m`
- **方法**:  
  - `fetchCommentsWithResourceId:resourceType:limit:offset:before:success:failure:` 主评论列表
  - `fetchFloorCommentsWithParentCommentId:resourceId:resourceType:limit:time:success:failure:` 楼层回复
- **用途**: 获取歌曲/专辑/歌单等资源的评论及回复

---

## 四、登录

### 9. NLGuestLoginService
- **路径**: `Networking/网易云API/Login/NLGuestLoginService.h/.m`
- **方法**: `anonymousLoginWithSuccess:failure:`
- **用途**: 游客匿名登录

### 10. NLPhoneLoginService
- **路径**: `Networking/网易云API/Login/NLPhoneLoginService.h/.m`
- **方法**:  
  - `loginWithPhone:password:success:failure:` 手机号密码登录
  - `loginWithPhone:captcha:success:failure:` 手机号验证码登录
  - `sendCaptchaWithPhone:success:failure:` 发送验证码
  - `verifyCaptchaWithPhone:captcha:success:failure:` 校验验证码
- **用途**: 手机号相关登录与验证

---

## 五、关系小结

| 场景       | 使用的 Service                         |
|------------|----------------------------------------|
| 主页-为你推荐 | NLRecommendAlbumListService |
| 主页-华语精品/欧美/流行 | NLChineseSongListService.fetchHighQualityPlaylistsWithCategory |
| 主页-歌手专辑 | NLSingerAlbumListService               |
| 歌单详情页 | NLSongListService                      |
| 专辑详情页 | NLAlbumService                         |
| 搜索-精选歌单 | NLPlaylistService                      |
| 播放音乐   | NLSongService                          |
| 评论区     | NLCommentService                       |

**说明**：`NLPlaylistService` 与 `NLChineseSongListService` 均调用 `/top/playlist/highquality`，均支持传入 cat 分类（华语、欧美、流行等）。
